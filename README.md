## Part 1

This part consists of a disassembler for the 8086 processor instruction set, as well as the simulation of some instructions, and cycle estimation for the test programs. The instruction set and number of cycles per instruction is described in the [Intel 8086 Users Manual](https://edge.edx.org/c4x/BITSPilani/EEE231/asset/8086_family_Users_Manual_1_.pdf).

There is an example command of how to run each .bat script at the bottom of each script.

>NOTE: The output of the part that simulates the programs cannot be diffed with the provided output .txt files from the course, because the formatting is different.
